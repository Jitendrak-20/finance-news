# Prompts

## Generation prompt intent

- original, neutral financial summary
- no fabricated facts
- no advice language
- separate headline, excerpt, body, SEO title, SEO description

## Validation checks

- block phrases such as `buy now`, `best stock`, `guaranteed gain`, `strong buy`, `sell now`
- require source attribution
- require a `Why this matters` section
- reject thin drafts that lack useful context
